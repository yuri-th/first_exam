<p>１．アプリケーション:Atte</p>
（概要)<br>
企業の勤怠管理システム

![atte-top](https://github.com/yuri-th/first_exam/assets/117786989/5bc75e36-d01b-4aef-96b6-eae6d0ecb15c)


<p>２．制作目的</p>
人事評価のため

<p>３．機能一覧</p>
・会員登録<br>
・ログイン、ログアウト<br>
・勤務開始、勤務終了<br>
※日付がかわったら翌日の出勤操作にきりかえる。<br>

・休憩開始、休憩終了<br>
※1日に何回でも休憩可能

・日付別勤怠情報取得<br>
・ページネーション（5件ずつ取得）<br>

<p>４．使用技術</p>
・Laravel Framework 8.83.27
